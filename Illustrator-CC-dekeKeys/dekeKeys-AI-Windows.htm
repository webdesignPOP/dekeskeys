dekeKeys for Illustrator: Windows

All shortcuts appear as listed in the Keyboard Shortcuts dialog box.
Default shortcuts in black. Custom dekeKeys shortcuts in red.
Illustrator shortcuts cannot include the Windows key
or the menu key (aka, the application key).
